Цель

Зачем нужен MOEX AI Hub

Проблема

AI выбран стратегическим направлением. Сотрудникам нужен единый доступ к промптам, скиллам и инструкциям без разрозненных источников.

Решение

Внутренняя платформа: библиотека промптов и скиллов, обучение, лидерборд и возможность делиться своими материалами.

Разделы платформы

Ключевые возможности

Библиотека

Промпты и скиллы с фильтрами по продукту (MOEX GPT, MOEX Insight, Code Agent), поиск, копирование в один клик.

Обучение

Гайды по установке и настройке, AI-тест на знание терминов, начисление mops за прохождение.

Лидерборд

Рейтинг активных пользователей по mops — геймификация и вовлечённость.

Подать материал

Сотрудники могут предложить свой промпт или скилл в общую библиотеку.

Технологии

Стек и доставка

Frontend: Next.js 14+ (App Router), TypeScript, Tailwind CSS, Shadcn/UI, Lucide. Интерфейс на русском языке.

Backend: FastAPI, PostgreSQL, Alembic. API для авторизации, квизов, пользователей. Готовность к деплою (Docker).

Структура проекта

Как устроено приложение

Полноценное веб-приложение с фронтендом и бэкендом, готовое к развёртыванию.

Маршруты (Frontend)

  • / — главная (hero, продукты, новые материалы, квиз)
  • /library — библиотека промптов и скиллов
  • /learn — обучение, гайды, глоссарий, AI-тест
  • /leaderboard — рейтинг по mops
  • /submit — подать свой материал

Backend API

  • /api/auth — авторизация
  • /api/users — пользователи
  • /api/quiz — квизы и ответы
  • PostgreSQL + Alembic (миграции)
  • Docker / docker-compose для деплоя
