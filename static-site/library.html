Основы

Промпт, скилл и MCP — что это?

Три формата, которые помогут работать эффективнее. Проверить знания можно в разделе Обучение — пройдите тест и проверьте свои знания.

Промпт

Текстовая инструкция для AI — описываете задачу, контекст и формат ответа.

Скилл

Custom Mode или Rules-файл для Code Agent — расширяет возможности AI-ассистента.

MCP-сервер

Конфигурация для подключения внешних инструментов (Jira, Git, БД) к Code Agent.
